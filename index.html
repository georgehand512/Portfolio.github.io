Meta-Assembler

Meta Assembler (written in C++) for use with my Cedar Logic based processor. The assembler reads a Cedar Logic Instruction Set reference from a Instruction_Set.txt file located in the Resource Files subfolder. The assembler then reads the “Users” East68K assembly program code from Assembly_Program.txt file in the same folder. The program then cross references these two object arrays to develop output Cedar Logic OpCodes. This output OpCode file is then output to the “Cedar Logic Ram Memory File.cdm” ready for upload into the Cedar logic processor. The user receives visual display of the conversion process, including error checking and messages.
